A simple adventure game by Pavithra Karthikeyan based on a simple adventure game engine by [Adam Smith](https://github.com/rndmcnlly).

Code requirements:
- **4+ scenes based on `AdventureScene`**: Room1, Room2, Room3, SecretRoom
- **2+ scenes *not* based on `AdventureScene`**: Intro, Outro
- **2+ methods or other enhancement added to the adventure game engine to simplify my scenes**:
    - Enhancement 1: describe (adds given pointerover text to a certain item)
    - Enhancement 2: pickupAnimation (adds an animation to an object to move away and disappear as though picked up)
Experience requirements:
- **4+ locations in the game world**: 3 main rooms of the gummy worm labs + secret rooms
- **2+ interactive objects in most scenes**: doors between rooms, notes on gummy worms
- **Many objects have `pointerover` messages**: gummy worm ingredients, notes on gummy worms
- **Many objects have `pointerdown` effects**: doors between rooms, key to secret room
- **Some objects are themselves animated**: keys, objects that can't be taken by the player (ex. gummy worm sugar jar)
